Safety Monitoring System

Developed a Safety Monitoring Webpage designed to enhance safety at oil and gas facilities. This responsive HTML-based interface, hosted on GitHub Pages, streamlines task-related information collection. Leveraging GPS, it automatically populates work location, weather details, and provides directions to nearby emergency facilities such as police, fire stations, and hospitals. The page intelligently autofills supervisor and safety officer details based on the job category (e.g., automation, electrical, mechanical). A standout feature is its webcam functionality, which detects whether a user is wearing a hardhat to ensure compliance with PPE standards. The captured image is processed using a YOLO Machine Learning Model I developed and trained, hosted on a Render server, providing real-time safety validation. All collected details are automatically stored in a Google Sheet using Apps Script, which is configured to send the data weekly via email through the Sheet Automation extension.
