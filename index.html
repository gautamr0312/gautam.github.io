Machine Learning enabled Back Exoskeleton

This project focused on developing a real-time, wearable assistive system designed to reduce lumbar stress for individuals in physically demanding industries such as construction and logistics. The goal was to predict lumbar flexion moments during lifting tasks and control an exoskeleton actuator to provide targeted support.


Key highlights:

  • Processed marker-based motion capture data from 10 subjects using Vicon Nexus software to capture full-body kinematics
  • Performed inverse kinematics and inverse dynamics analysis in OpenSim to estimate ground-truth lumbar flexion moments for model training
  • Deployed a Temporal Convolutional Neural Network (TCN) model on an NVIDIA Jetson Nano to predict lumbar moments in real-time, using data from a nine-axis MicroStrain IMU and integrated motor sensors
  • Engineered a velocity controller on an Arduino Teensy to perform closed-loop motor torque control, dynamically adjusting exoskeleton assistance based on the predicted lumbar loading
  • Designed the system to enhance safety and reduce injury risk for workers performing repetitive or heavy lifting tasks

This project combined wearable sensor integration, real-time embedded control, and machine learning for human biomechanics prediction, contributing to the development of intelligent assistive exoskeleton technology for industrial applications.

This project enhanced my skills in deep learning, computer vision techniques, and working with real-world, imperfect datasets.

Image Captioning

In this project, I developed a deep learning model capable of generating descriptive captions for images by combining computer vision and natural language processing techniques. The system uses a pre-trained CNN (ResNet-50) to extract image features, which are then fed into an LSTM-based RNN decoder to generate captions word-by-word.


Key highlights:

  • Created a custom vocabulary and handled image-caption pair preprocessing
  • Built an Encoder-Decoder architecture using PyTorch
  • Implemented techniques like hyperparameter tuning and transfer learning for improved model performance
  • Evaluated model output qualitatively

This project strengthened my understanding of image-to-sequence problems and multi-modal deep learning architectures.

Safety Monitoring Mobile Application for Oil & Gas Facilities

To enhance on-site safety and streamline operations in oil and gas facilities, I developed a responsive, mobile-friendly Safety Monitoring Webpage, hosted on GitHub Pages. Designed for field technicians and safety officers, this application automates the collection of essential task-related data while ensuring compliance with PPE (Personal Protective Equipment) standards.


Key highlights:

  • Automatic GPS integration to detect and populate work location, current weather conditions, and provide real-time directions to the nearest emergency facilities, including police stations, fire departments, and hospitals
  • Smart autofill functionality for supervisor and safety officer details based on the selected job category (e.g., automation, electrical, mechanical)
  • An integrated webcam-based hardhat detection system that captures live images and verifies PPE compliance using a custom-trained YOLO (You Only Look Once) Machine Learning model, hosted on a Render server
  • Seamless data management where all collected information is automatically stored in a Google Sheet via Google Apps Script, with automated weekly report emails sent through the Sheet Automation extension

This project combined my skills in Computer Vision, Web Development, and Cloud Deployment, delivering a practical safety solution tailored for industrial environments.

Integration of Multi-Modal Sensing & Navigation for Autonomous Maze Navigation

In this robotics project, I developed an autonomous navigation system for a TurtleBot capable of detecting traffic signs, making decisions based on environmental cues, and navigating through a maze environment under time constraints.


Key highlights:

  • Developed a sequential neural network model achieving 92.6% accuracy in traffic sign detection, enabling the robot to recognize critical navigation commands such as stop, turn, and speed adjustments
  • Implemented a finite state machine (FSM) for decision-making that integrated neural network predictions with data from lidar and camera, allowing the robot to dynamically adjust its behavior in response to both visual and sensor inputs
  • Designed and implemented angular and linear velocity controllers to manage the robot’s movement, ensuring smooth and accurate navigation through the maze
  • Successfully optimized the system to complete maze navigation tasks in under 180 seconds, demonstrating efficient sensor fusion, control, and real-time decision-making in a constrained environment

This project combined robot perception, state-based control systems, sensor integration, and autonomous navigation, reinforcing my skills in developing intelligent mobile robot applications.

Detection and Classification Using LiDAR

This project focused on developing and evaluating object classification techniques on LiDAR point cloud images, with the goal of identifying optimal models for efficient perception in robotics and autonomous systems.


Key highlights:

  • Processed a dataset of 3,000 LiDAR-generated images and applied image preprocessing techniques, including adaptive histogram equalization, to enhance image contrast and feature visibility
  • Compared the performance of multiple classification algorithms, including a Haar Cascade Classifier, K-Nearest Neighbors (KNN), Support Vector Machine (SVM), and a YOLOv5-nano deep learning model
  • Achieved the best results using YOLOv5-nano, obtaining 89% accuracy, 75% recall, and 75% precision on the test set after optimized image processing
  • Conducted detailed performance evaluation to benchmark traditional machine learning classifiers against modern real-time object detection models on LiDAR data

This project reinforced my skills in computer vision, LiDAR data processing, model evaluation, and real-time object detection frameworks for robotics and perception applications.

This project combined mechanical design, embedded systems integration, and human-centered engineering to address real-world mobility challenges with an emphasis on flexibility and user safety.
